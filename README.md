## ▶️ How to Run in VS Code

### Step 1 — Open Workspace
Double-click `CodeAlpha.code-workspace`  
OR in VS Code: **File → Open Workspace from File…** → select `CodeAlpha.code-workspace`

### Step 2 — Install Live Server (one-time)
1. Press `Ctrl+Shift+X` to open Extensions
2. Search **"Live Server"** by Ritwick Dey
3. Click **Install**

### Step 3 — Run Any Project
1. Open the `index.html` file of any task
2. Right-click anywhere in the file
3. Click **"Open with Live Server"**
4. Browser opens at `http://127.0.0.1:5500`

## 💡 Tips

- Each task is **fully self-contained** — just one `index.html` file, no dependencies to install
- All projects use **Google Fonts** loaded from CDN — requires internet connection
- Keyboard shortcuts work in Gallery (`←` `→` `Esc`) and Calculator (all keys)
